Comandi da eseguire nella root del progetto
1) Generazione del file "target/smarcare-0.0.1-SNAPSHOT.jar" ./mvnw clean package
2) Avvio dell'applicazione ./mvnw spring-boot:run
3) Avviare il container con dentro l'immagine mongo docker-compose -f docker-compose.yml up -d mongo

git repo del progetto: https://github.com/badluck1024/smarcare
